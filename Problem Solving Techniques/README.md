# Problem-Solving Techniques (Part 2)

Where [Maths](../Maths/README.md) built the mathematical foundations (arithmetic, number theory, combinatorics, and so on), this section is about **patterns of thinking** — the recurring strategies that turn a hard problem into a solvable one. Each lesson covers one technique: what it is, when to reach for it, its common variants, and worked Java examples.

Every lesson follows the same structure as Part 1:

- **Theory** — what the technique is and when it applies.
- **Variants** — the different flavors the technique shows up in.
- **Code** — working Java implementations for each variant.
- **Example** — sample input, a dry run, and the output.

> **Language used:** Java
