## 1. Distance

### Theory

The distance between two points `(x1, y1)` and `(x2, y2)` comes directly from the Pythagorean theorem — the straight-line ("Euclidean") distance is the hypotenuse of the right triangle formed by the horizontal and vertical gaps:

```
distance = √((x2-x1)² + (y2-y1)²)
```

### Code

```java
public class Distance {
    public static void main(String[] args) {
        System.out.println("Distance (0,0) to (3,4): " + distance(0, 0, 3, 4));
    }

    static double distance(double x1, double y1, double x2, double y2) {
        double dx = x2 - x1;
        double dy = y2 - y1;
        return Math.sqrt(dx * dx + dy * dy);
    }
}
```

### Example

```
Distance (0,0) to (3,4): 5.0
```

The classic 3-4-5 right triangle.

## 2. Slope

### Theory

The slope of the line through `(x1, y1)` and `(x2, y2)` measures steepness — "rise over run":

```
slope = (y2 - y1) / (x2 - x1)
```

> **Vertical line gotcha:** when `x1 == x2`, the line is vertical and the slope is **undefined** (division by zero) — this must be special-cased, not computed.

### Code

```java
public class Slope {
    public static void main(String[] args) {
        System.out.println("Slope (1,2) to (3,6): " + slope(1, 2, 3, 6));
        System.out.println("Slope (2,1) to (2,5): " + slope(2, 1, 2, 5)); // vertical line
    }

    static Double slope(double x1, double y1, double x2, double y2) {
        if (x1 == x2) return null; // vertical line, slope is undefined
        return (y2 - y1) / (x2 - x1);
    }
}
```

### Example

```
Slope (1,2) to (3,6): 2.0
Slope (2,1) to (2,5): null
```

## 3. Collinearity

### Theory

Three points are **collinear** if they lie on the same straight line. Rather than computing slopes (which breaks down for vertical lines), the standard test uses the **cross product** of the vectors from the first point to the other two — it's zero exactly when the three points are collinear:

```
cross = (x2-x1)(y3-y1) - (y2-y1)(x3-x1)
collinear  iff  cross == 0
```

### Code

```java
public class Collinearity {
    public static void main(String[] args) {
        System.out.println("(1,1),(2,2),(3,3) collinear: " + areCollinear(1, 1, 2, 2, 3, 3));
        System.out.println("(1,1),(2,3),(3,3) collinear: " + areCollinear(1, 1, 2, 3, 3, 3));
    }

    static boolean areCollinear(double x1, double y1, double x2, double y2, double x3, double y3) {
        double cross = (x2 - x1) * (y3 - y1) - (y2 - y1) * (x3 - x1);
        return cross == 0;
    }
}
```

### Example

```
(1,1),(2,2),(3,3) collinear: true
(1,1),(2,3),(3,3) collinear: false
```

## 4. Area

### Theory

The area of a triangle given its three vertices' coordinates uses the same cross-product idea as collinearity — it's exactly half the absolute value of that cross product (which is zero for collinear "triangles" with no area, consistent with Section 3):

```
area = |x1(y2-y3) + x2(y3-y1) + x3(y1-y2)| / 2
```

This is the `n = 3` case of the general **Shoelace Formula** covered in [Polygon](#9-polygon) below.

### Code

```java
public class Area {
    public static void main(String[] args) {
        System.out.println("Area of triangle (0,0),(4,0),(0,3): " + triangleArea(0, 0, 4, 0, 0, 3));
    }

    static double triangleArea(double x1, double y1, double x2, double y2, double x3, double y3) {
        return Math.abs(x1 * (y2 - y3) + x2 * (y3 - y1) + x3 * (y1 - y2)) / 2.0;
    }
}
```

### Example

```
Area of triangle (0,0),(4,0),(0,3): 6.0
```

## 5. Perimeter

### Theory

The perimeter of any polygon (or just a path of points) is the sum of the [distances](#1-distance) between each pair of **consecutive** vertices, wrapping back from the last vertex to the first.

### Code

```java
public class Perimeter {
    public static void main(String[] args) {
        double[][] triangle = { { 0, 0 }, { 4, 0 }, { 0, 3 } };
        System.out.println("Perimeter of triangle: " + perimeter(triangle));
    }

    static double perimeter(double[][] points) {
        int n = points.length;
        double total = 0;
        for (int i = 0; i < n; i++) {
            double[] current = points[i];
            double[] next = points[(i + 1) % n]; // wrap around to the first point
            total += distance(current[0], current[1], next[0], next[1]);
        }
        return total;
    }

    static double distance(double x1, double y1, double x2, double y2) {
        double dx = x2 - x1, dy = y2 - y1;
        return Math.sqrt(dx * dx + dy * dy);
    }
}
```

### Example

```
Perimeter of triangle: 12.0
```

The triangle `(0,0), (4,0), (0,3)` has sides of length `4`, `5`, and `3` — a 3-4-5 right triangle, perimeter `12`.

## 6. Rectangle

### Theory

For an axis-aligned rectangle with width `w` and height `h`:

```
area      = w × h
perimeter = 2 × (w + h)
```

### Code

```java
public class Rectangle {
    public static void main(String[] args) {
        double width = 5, height = 3;
        System.out.println("Rectangle area: " + area(width, height));
        System.out.println("Rectangle perimeter: " + perimeter(width, height));
    }

    static double area(double width, double height) {
        return width * height;
    }

    static double perimeter(double width, double height) {
        return 2 * (width + height);
    }
}
```

### Example

```
Rectangle area: 15.0
Rectangle perimeter: 16.0
```

## 7. Circle

### Theory

For a circle with radius `r`:

```
area          = π × r²
circumference = 2 × π × r
```

Java's `Math.PI` gives the constant to full `double` precision.

### Code

```java
public class Circle {
    public static void main(String[] args) {
        double radius = 7;
        System.out.println("Circle area: " + area(radius));
        System.out.println("Circle circumference: " + circumference(radius));
    }

    static double area(double radius) {
        return Math.PI * radius * radius;
    }

    static double circumference(double radius) {
        return 2 * Math.PI * radius;
    }
}
```

### Example

```
Circle area: 153.93804002589985
Circle circumference: 43.982297150257104
```

## 8. Triangle

### Theory

When you have **side lengths** instead of coordinates, **Heron's Formula** computes the area directly — no angles or coordinates needed:

```
s = (a + b + c) / 2                    (semi-perimeter)
area = √(s × (s-a) × (s-b) × (s-c))
```

### Code

```java
public class Triangle {
    public static void main(String[] args) {
        System.out.println("Area of 3-4-5 triangle (Heron's formula): " + heronArea(3, 4, 5));
    }

    static double heronArea(double a, double b, double c) {
        double s = (a + b + c) / 2;
        return Math.sqrt(s * (s - a) * (s - b) * (s - c));
    }
}
```

### Example

```
Area of 3-4-5 triangle (Heron's formula): 6.0
```

Matches the coordinate-based triangle area from [Section 4](#4-area) — both describe the same 3-4-5 right triangle.

## 9. Polygon

### Theory

The **Shoelace Formula** generalizes the triangle-area trick (Section 4) to any simple polygon with `n` vertices listed in order (clockwise or counterclockwise):

```
area = |Σ (xᵢ × yᵢ₊₁ - xᵢ₊₁ × yᵢ)| / 2      (indices wrap around: vertex n connects back to vertex 0)
```

### Code

```java
public class Polygon {
    public static void main(String[] args) {
        double[][] square = { { 0, 0 }, { 4, 0 }, { 4, 4 }, { 0, 4 } };
        System.out.println("Area of 4x4 square via Shoelace Formula: " + shoelaceArea(square));
    }

    static double shoelaceArea(double[][] points) {
        int n = points.length;
        double sum = 0;
        for (int i = 0; i < n; i++) {
            double[] current = points[i];
            double[] next = points[(i + 1) % n];
            sum += current[0] * next[1] - next[0] * current[1];
        }
        return Math.abs(sum) / 2.0;
    }
}
```

### Example

```
Area of 4x4 square via Shoelace Formula: 16.0
```

## 10. Coordinate Geometry

### Theory

The cross product from [Collinearity](#3-collinearity) doesn't just detect "on the same line" — its **sign** tells you which way the path from point `A` to `B` to `C` turns:

- Positive → **counterclockwise** turn
- Negative → **clockwise** turn
- Zero → **collinear** (no turn at all)

This "orientation test" is a fundamental building block for computational geometry algorithms (convex hull, line segment intersection, point-in-polygon tests).

### Code

```java
public class CoordinateGeometry {
    public static void main(String[] args) {
        System.out.println("Orientation of (0,0),(4,0),(4,4): " + orientation(0, 0, 4, 0, 4, 4));
        System.out.println("Orientation of (0,0),(4,4),(4,0): " + orientation(0, 0, 4, 4, 4, 0));
    }

    static String orientation(double x1, double y1, double x2, double y2, double x3, double y3) {
        double cross = (x2 - x1) * (y3 - y1) - (y2 - y1) * (x3 - x1);
        if (cross == 0) return "collinear";
        return cross > 0 ? "counterclockwise" : "clockwise";
    }
}
```

### Example

```
Orientation of (0,0),(4,0),(4,4): counterclockwise
Orientation of (0,0),(4,4),(4,0): clockwise
```

## Quick Reference Cheat Sheet

| Concept | Formula |
|---|---|
| Distance | `√((x2-x1)² + (y2-y1)²)` |
| Slope | `(y2-y1)/(x2-x1)`, undefined if `x1 == x2` |
| Collinearity | cross product of the two vectors `== 0` |
| Triangle Area (coords) | `\|x1(y2-y3) + x2(y3-y1) + x3(y1-y2)\| / 2` |
| Perimeter | sum of consecutive-vertex distances, wrapping around |
| Rectangle | area `= w×h`, perimeter `= 2(w+h)` |
| Circle | area `= πr²`, circumference `= 2πr` |
| Triangle Area (sides) | Heron's formula: `√(s(s-a)(s-b)(s-c))` |
| Polygon Area | Shoelace Formula, generalizes triangle area to `n` vertices |
| Orientation | sign of cross product: `+` CCW, `-` CW, `0` collinear |
