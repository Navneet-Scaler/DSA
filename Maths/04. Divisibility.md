## 1. Even/Odd

### Theory

A number is **even** if it's exactly divisible by 2 (`n % 2 == 0`), and **odd** otherwise. There's also a faster bitwise check: the last bit of a binary number tells you its parity — `n & 1` is `0` for even numbers and `1` for odd numbers, since only the least significant bit determines whether a number is a multiple of 2.

> Bitwise `&` works correctly for negative numbers too in Java (two's complement keeps the last bit meaningful), so `n & 1` is a safe, allocation-free alternative to `%` in tight loops.

### Code

```java
public class EvenOdd {
    public static void main(String[] args) {
        int a = 18;
        int b = 17;

        System.out.println(a + " is even (a % 2 == 0): " + (a % 2 == 0));
        System.out.println(b + " is even (b % 2 == 0): " + (b % 2 == 0));

        // Bitwise check
        System.out.println(a + " is even (a & 1) == 0: " + ((a & 1) == 0));
        System.out.println(b + " is even (b & 1) == 0: " + ((b & 1) == 0));
    }
}
```

### Example

```
18 is even (a % 2 == 0): true
17 is even (b % 2 == 0): false
18 is even (a & 1) == 0: true
17 is even (b & 1) == 0: false
```

## 2. Divisibility Tests

### Theory

The direct way to test if `a` is divisible by `b` is `a % b == 0`. But several classic **digit-based shortcuts** let you test divisibility by small numbers without doing full division — useful for mental math and for problems that only give you a number as a string of digits (too large for `int`/`long`):

| Divisor | Rule |
|---|---|
| 2 | Last digit is even |
| 3 | Sum of digits divisible by 3 |
| 4 | Last two digits form a number divisible by 4 |
| 5 | Last digit is 0 or 5 |
| 6 | Divisible by both 2 and 3 |
| 8 | Last three digits divisible by 8 |
| 9 | Sum of digits divisible by 9 |
| 10 | Last digit is 0 |
| 11 | Alternating sum of digits (from the right) divisible by 11 |

### Code

```java
public class DivisibilityTests {
    public static void main(String[] args) {
        int n = 132;

        System.out.println(n + " divisible by 2 (direct):   " + (n % 2 == 0));
        System.out.println(n + " divisible by 3 (direct):   " + (n % 3 == 0));

        // Digit-sum rule for 3 and 9
        int digitSum = 0;
        int temp = n;
        while (temp != 0) {
            digitSum += temp % 10;
            temp /= 10;
        }
        System.out.println(n + " digit sum: " + digitSum);
        System.out.println(n + " divisible by 3 (digit-sum rule): " + (digitSum % 3 == 0));
        System.out.println(n + " divisible by 9 (digit-sum rule): " + (digitSum % 9 == 0));

        // Last-two-digits rule for 4
        int lastTwo = n % 100;
        System.out.println(n + " divisible by 4 (last two digits = " + lastTwo + "): " + (lastTwo % 4 == 0));

        // Alternating-sum rule for 11
        int alternating = 0;
        int sign = 1;
        temp = n;
        while (temp != 0) {
            alternating += sign * (temp % 10);
            sign *= -1;
            temp /= 10;
        }
        System.out.println(n + " alternating digit sum: " + alternating);
        System.out.println(n + " divisible by 11 (alternating-sum rule): " + (alternating % 11 == 0));
    }
}
```

### Example

```
132 divisible by 2 (direct):   true
132 divisible by 3 (direct):   true
132 digit sum: 6
132 divisible by 3 (digit-sum rule): true
132 divisible by 9 (digit-sum rule): false
132 divisible by 4 (last two digits = 32): true
132 alternating digit sum: 0
132 divisible by 11 (alternating-sum rule): true
```

## 3. Multiples

### Theory

A **multiple** of `n` is any number obtained by multiplying `n` by an integer (`n, 2n, 3n, ...`). Checking whether `a` is a multiple of `b` is the same test as divisibility: `a % b == 0`. Generating the first `k` multiples of `n` is a simple loop.

### Code

```java
public class Multiples {
    public static void main(String[] args) {
        int n = 4;
        int count = 5;

        System.out.print("First " + count + " multiples of " + n + ": ");
        for (int i = 1; i <= count; i++) {
            System.out.print((n * i) + " ");
        }
        System.out.println();

        // Checking if a number is a multiple of another
        int a = 20;
        System.out.println(a + " is a multiple of " + n + ": " + (a % n == 0));
    }
}
```

### Example

```
First 5 multiples of 4: 4 8 12 16 20
20 is a multiple of 4: true
```

## 4. Factors

### Theory

A **factor** (or **divisor**) of `n` is any number that divides it exactly (remainder `0`). The naive way checks every number from `1` to `n` — O(n). The efficient way only checks up to `√n`: whenever `i` divides `n`, both `i` and `n / i` are factors, so you can collect **pairs** and stop at the square root — O(√n).

```
if i divides n:
    factors found: i  AND  n / i
```

### Code

```java
import java.util.TreeSet;

public class Factors {
    public static void main(String[] args) {
        int n = 36;
        TreeSet<Integer> factors = new TreeSet<>();

        for (int i = 1; (long) i * i <= n; i++) {
            if (n % i == 0) {
                factors.add(i);
                factors.add(n / i);
            }
        }

        System.out.println("Factors of " + n + ": " + factors);
    }
}
```

### Example

```
Factors of 36: [1, 2, 3, 4, 6, 9, 12, 18, 36]
```

## 5. Proper Factors

### Theory

**Proper factors** of `n` are all its factors *excluding `n` itself* (so `1` is included, but `n` is not). They're the building block for perfect-number checks and for computing a number's "aliquot sum".

### Code

```java
import java.util.TreeSet;

public class ProperFactors {
    public static void main(String[] args) {
        int n = 36;
        TreeSet<Integer> properFactors = new TreeSet<>();

        for (int i = 1; (long) i * i <= n; i++) {
            if (n % i == 0) {
                if (i != n) properFactors.add(i);
                int pair = n / i;
                if (pair != n) properFactors.add(pair);
            }
        }

        int sum = 0;
        for (int f : properFactors) sum += f;

        System.out.println("Proper factors of " + n + ": " + properFactors);
        System.out.println("Sum of proper factors: " + sum);
    }
}
```

### Example

```
Proper factors of 36: [1, 2, 3, 4, 6, 9, 12, 18]
Sum of proper factors: 55
```

## 6. Perfect Divisors (Perfect Numbers)

### Theory

A **perfect number** is a number that equals the **sum of its own proper factors**. The classic small example is `28`: its proper factors are `1, 2, 4, 7, 14`, and `1 + 2 + 4 + 7 + 14 = 28`. Perfect numbers are rare — the next ones after `6` and `28` are `496` and `8128`.

> A number whose proper-factor sum is **less** than itself is called **deficient**; one where it's **greater** is called **abundant**. Perfect numbers sit exactly on the boundary.

### Code

```java
public class PerfectNumber {
    public static void main(String[] args) {
        System.out.println("6 is perfect: " + isPerfect(6));
        System.out.println("28 is perfect: " + isPerfect(28));
        System.out.println("12 is perfect: " + isPerfect(12));
    }

    static boolean isPerfect(int n) {
        int sum = 0;
        for (int i = 1; (long) i * i <= n; i++) {
            if (n % i == 0) {
                if (i != n) sum += i;
                int pair = n / i;
                if (pair != i && pair != n) sum += pair;
            }
        }
        return sum == n;
    }
}
```

### Example

```
6 is perfect: true
28 is perfect: true
12 is perfect: false
```

## 7. Common Divisors

### Theory

A **common divisor** of two numbers is any value that divides both exactly. The set of common divisors is the **intersection** of each number's factor set, and the largest one is the **GCD (Greatest Common Divisor)** — normally computed efficiently with the **Euclidean algorithm** rather than by intersecting full factor lists:

```
gcd(a, b) = gcd(b, a % b),   until b == 0, then gcd = a
```

### Code

```java
import java.util.TreeSet;

public class CommonDivisors {
    public static void main(String[] args) {
        int a = 36;
        int b = 24;

        TreeSet<Integer> factorsA = allFactors(a);
        TreeSet<Integer> factorsB = allFactors(b);

        TreeSet<Integer> common = new TreeSet<>(factorsA);
        common.retainAll(factorsB);

        System.out.println("Factors of " + a + ": " + factorsA);
        System.out.println("Factors of " + b + ": " + factorsB);
        System.out.println("Common divisors: " + common);
        System.out.println("Greatest common divisor (Euclidean): " + gcd(a, b));
    }

    static TreeSet<Integer> allFactors(int n) {
        TreeSet<Integer> factors = new TreeSet<>();
        for (int i = 1; (long) i * i <= n; i++) {
            if (n % i == 0) {
                factors.add(i);
                factors.add(n / i);
            }
        }
        return factors;
    }

    static int gcd(int a, int b) {
        while (b != 0) {
            int temp = b;
            b = a % b;
            a = temp;
        }
        return a;
    }
}
```

### Example

```
Factors of 36: [1, 2, 3, 4, 6, 9, 12, 18, 36]
Factors of 24: [1, 2, 3, 4, 6, 8, 12, 24]
Common divisors: [1, 2, 3, 4, 6, 12]
Greatest common divisor (Euclidean): 12
```

## Quick Reference Cheat Sheet

| Concept | Core check / formula |
|---|---|
| Even / Odd | `n % 2 == 0`, or `(n & 1) == 0` |
| Divisibility by 3 / 9 | digit sum divisible by 3 / 9 |
| Divisibility by 4 / 8 | last 2 / 3 digits divisible by 4 / 8 |
| Divisibility by 11 | alternating digit sum divisible by 11 |
| Multiple check | `a % b == 0` |
| Factors of `n` | loop `i` from `1` to `√n`, collect `i` and `n/i` |
| Proper factors | factors of `n` excluding `n` itself |
| Perfect number | sum of proper factors `== n` |
| Common divisors | intersection of both numbers' factor sets |
| GCD | Euclidean algorithm: `gcd(a, b) = gcd(b, a % b)` |
