# Maths for DSA

Mathematics is the backbone of Data Structures & Algorithms — it shows up in complexity analysis, number theory problems, bit manipulation, probability-based algorithms, and geometry problems. This section builds the foundation from the ground up.

Each lesson follows the same three-part structure:

- **Theory** — the concept explained simply, with rules and edge cases.
- **Code** — a working Java implementation.
- **Example** — sample input, a dry run, and the output.
