# 3. Character Operations

Most string algorithms bottom out in per-character checks and conversions — is this a digit, is it uppercase, what's its numeric value? Java's `Character` class provides static utility methods for all of it, mirroring `Math`'s role for numbers (see [Maths Lesson 26](<../Maths/26. Java-Specific Math.md>)).

## 1. The Character Class

### Theory

`java.lang.Character` is a **final class of static methods**, exactly like `Math` — it can't be instantiated, and every classification/conversion method is called directly on the class (`Character.isDigit(c)`). It also wraps the `char` primitive as an object (`Character c = 'a';` via autoboxing) when a reference type is needed, e.g. in generic collections.

---

## 2. isDigit(), isLetter() & isLetterOrDigit()

### Theory

Three classification checks, each testing progressively broader Unicode categories: `isDigit()` checks for a decimal digit (in *any* script, not just `0-9` — Java correctly recognizes Arabic-Indic and other digit systems too), `isLetter()` checks for an alphabetic character, and `isLetterOrDigit()` is their union — the common "is this an alphanumeric character?" check.

### Code

```java
public class DigitLetterChecks {
    public static void main(String[] args) {
        System.out.println("isDigit('5'): " + Character.isDigit('5'));
        System.out.println("isLetter('a'): " + Character.isLetter('a'));
        System.out.println("isLetterOrDigit('!'): " + Character.isLetterOrDigit('!'));
        System.out.println("isLetterOrDigit('a'): " + Character.isLetterOrDigit('a'));
    }
}
```

### Example

```
isDigit('5'): true
isLetter('a'): true
isLetterOrDigit('!'): false
isLetterOrDigit('a'): true
```

## 3. isUpperCase() & isLowerCase()

### Theory

Check a character's case directly — useful for validation and for case-sensitive logic without needing to convert first.

### Code

```java
public class CaseChecks {
    public static void main(String[] args) {
        System.out.println("isUpperCase('A'): " + Character.isUpperCase('A'));
        System.out.println("isLowerCase('A'): " + Character.isLowerCase('A'));
    }
}
```

### Example

```
isUpperCase('A'): true
isLowerCase('A'): false
```

---

## 4. toUpperCase() & toLowerCase()

### Theory

Convert a single character's case. (The `String` versions, `"...".toUpperCase()`/`toLowerCase()`, apply this to every character at once and are locale-sensitive — a detail that occasionally matters for languages like Turkish, where `i`/`I` don't map the way English speakers expect.)

### Code

```java
public class CaseConversion {
    public static void main(String[] args) {
        System.out.println(Character.toUpperCase('a'));
        System.out.println(Character.toLowerCase('A'));
    }
}
```

### Example

```
A
a
```

## 6. Character Conversion

### Theory

Two conversions come up constantly:

1. **Digit character ↔ its numeric value** — since `'0'`-`'9'` are contiguous in Unicode (values 48-57), `c - '0'` converts a digit character to its `int` value, and `'0' + digit` converts back.
2. **`char` ↔ `int` (Unicode code point)** — a `char` widens to `int` implicitly (giving its raw code point, *not* a digit value unless it happens to be a digit character), and an `int` narrows back to `char` with an explicit cast.

`Character.getNumericValue(c)` is a safer general-purpose alternative that also handles letter-as-digit systems (`'A'` → `10`, useful for base-36 parsing) and returns `-1` for non-numeric input instead of an incorrect value.

### Code

```java
public class CharacterConversion {
    public static void main(String[] args) {
        char digitChar = '7';
        int digitValue = digitChar - '0';
        System.out.println("'7' - '0' = " + digitValue);

        int value = 9;
        char asChar = (char) ('0' + value);
        System.out.println("'0' + 9 (as char) = " + asChar);

        System.out.println("Character.getNumericValue('7'): " + Character.getNumericValue('7'));
        System.out.println("Character.getNumericValue('A'): " + Character.getNumericValue('A'));
    }
}
```

### Example

```
'7' - '0' = 7
'0' + 9 (as char) = 9
Character.getNumericValue('7'): 7
Character.getNumericValue('A'): 10
```

## Quick Reference Cheat Sheet

| Method | Purpose |
|---|---|
| `Character.isDigit(c)` | is `c` a decimal digit (any script) |
| `Character.isLetter(c)` | is `c` alphabetic |
| `Character.isLetterOrDigit(c)` | union of the above |
| `Character.isUpperCase(c)` / `isLowerCase(c)` | case check |
| `Character.toUpperCase(c)` / `toLowerCase(c)` | case conversion |
| `c - '0'` / `'0' + digit` | digit char ↔ numeric value |
| `Character.getNumericValue(c)` | safer digit/letter-as-digit conversion, `-1` if invalid |
