## 1. Suffix Array Construction

### Theory

A **suffix array** is the list of a string's starting indices, **sorted** by the suffix that begins there. `"banana"` has suffixes `"banana"`, `"anana"`, `"nana"`, `"ana"`, `"na"`, `"a"` — sorted alphabetically: `"a"`, `"ana"`, `"anana"`, `"banana"`, `"na"`, `"nana"`, giving the suffix array `[5, 3, 1, 0, 4, 2]`. The simplest construction just sorts all `n` suffixes directly using `String.compareTo` — **O(n² log n)** (each of the `n log n` comparisons can itself take O(n)) — good enough for small-to-medium strings and for building intuition before reaching for the O(n log n) or O(n) constructions used at scale.

### Code

```java
import java.util.Arrays;
import java.util.Integer;

public class SuffixArray {
    public static void main(String[] args) {
        int[] sa = buildSuffixArray("banana");
        System.out.println("Suffix array of 'banana': " + Arrays.toString(sa));

        String[] suffixesInOrder = new String[sa.length];
        for (int i = 0; i < sa.length; i++) suffixesInOrder[i] = "banana".substring(sa[i]);
        System.out.println("Suffixes in order: " + Arrays.toString(suffixesInOrder));
    }

    static int[] buildSuffixArray(String s) {
        int n = s.length();
        Integer[] indices = new Integer[n];
        for (int i = 0; i < n; i++) indices[i] = i;

        Arrays.sort(indices, (a, b) -> s.substring(a).compareTo(s.substring(b)));

        int[] suffixArray = new int[n];
        for (int i = 0; i < n; i++) suffixArray[i] = indices[i];
        return suffixArray;
    }
}
```

### Example

```
Suffix array of 'banana': [5, 3, 1, 0, 4, 2]
Suffixes in order: [a, ana, anana, banana, na, nana]
```

## 2. LCP Array and Longest Repeated Substring

### Theory

The **LCP (Longest Common Prefix) array** rides alongside the suffix array: `lcp[i]` is the length of the longest common prefix between the suffixes at sorted positions `i-1` and `i`. Once suffixes are sorted, any two suffixes that share a long common prefix end up **adjacent** in the sorted order — so the LCP array (built efficiently with **Kasai's algorithm**, which reuses the previous suffix's LCP to avoid recomputing from scratch each time) captures exactly the repeated-substring structure of the string. **Classic example — Longest Repeated Substring:** the answer is simply the suffix at the position of the **maximum** LCP value, truncated to that length.

### Code

```java
import java.util.Arrays;

public class LongestRepeatedSubstring {
    public static void main(String[] args) {
        System.out.println("Longest repeated substring of 'banana': " + longestRepeatedSubstring("banana"));
        System.out.println("Longest repeated substring of 'abcabcabc': " + longestRepeatedSubstring("abcabcabc"));
    }

    static String longestRepeatedSubstring(String s) {
        int n = s.length();
        int[] sa = buildSuffixArray(s);
        int[] lcp = buildLcpArray(s, sa);

        int maxLen = 0, bestIndex = 0;
        for (int i = 0; i < n; i++) {
            if (lcp[i] > maxLen) {
                maxLen = lcp[i];
                bestIndex = sa[i];
            }
        }
        return maxLen > 0 ? s.substring(bestIndex, bestIndex + maxLen) : "";
    }

    static int[] buildSuffixArray(String s) {
        int n = s.length();
        Integer[] indices = new Integer[n];
        for (int i = 0; i < n; i++) indices[i] = i;
        Arrays.sort(indices, (a, b) -> s.substring(a).compareTo(s.substring(b)));
        int[] sa = new int[n];
        for (int i = 0; i < n; i++) sa[i] = indices[i];
        return sa;
    }

    // Kasai's algorithm: builds the LCP array in O(n) given the suffix array
    static int[] buildLcpArray(String s, int[] sa) {
        int n = s.length();
        int[] rank = new int[n];
        for (int i = 0; i < n; i++) rank[sa[i]] = i;

        int[] lcp = new int[n];
        int h = 0;
        for (int i = 0; i < n; i++) {
            if (rank[i] > 0) {
                int j = sa[rank[i] - 1];
                while (i + h < n && j + h < n && s.charAt(i + h) == s.charAt(j + h)) h++;
                lcp[rank[i]] = h;
                if (h > 0) h--;
            } else {
                h = 0;
            }
        }
        return lcp;
    }
}
```

### Example

```
Longest repeated substring of 'banana': ana
Longest repeated substring of 'abcabcabc': abcabc
```

`"banana"`'s longest repeated substring is `"ana"` (appears at indices 1 and 3, overlapping). `"abcabcabc"`'s is `"abcabc"` — a repeated substring can itself contain smaller repeats.

## 3. Counting Distinct Substrings

### Theory

**Classic example — Count Distinct Substrings:** count how many **distinct** (non-duplicate) substrings a string has. A string of length `n` has `n(n+1)/2` total substrings **counting duplicates**; the LCP array directly measures the duplication — each `lcp[i]` counts exactly how many characters of the suffix at sorted position `i` are a **repeat** of a prefix already counted at position `i-1`. So the distinct count is simply the total minus the sum of the entire LCP array.

### Code

```java
import java.util.Arrays;

public class CountDistinctSubstrings {
    public static void main(String[] args) {
        System.out.println("Distinct substrings of 'aaa': " + countDistinctSubstrings("aaa"));
        System.out.println("Distinct substrings of 'abc': " + countDistinctSubstrings("abc"));
    }

    static long countDistinctSubstrings(String s) {
        int n = s.length();
        int[] sa = buildSuffixArray(s);
        int[] lcp = buildLcpArray(s, sa);

        long total = (long) n * (n + 1) / 2;
        for (int value : lcp) total -= value;
        return total;
    }

    static int[] buildSuffixArray(String s) {
        int n = s.length();
        Integer[] indices = new Integer[n];
        for (int i = 0; i < n; i++) indices[i] = i;
        Arrays.sort(indices, (a, b) -> s.substring(a).compareTo(s.substring(b)));
        int[] sa = new int[n];
        for (int i = 0; i < n; i++) sa[i] = indices[i];
        return sa;
    }

    static int[] buildLcpArray(String s, int[] sa) {
        int n = s.length();
        int[] rank = new int[n];
        for (int i = 0; i < n; i++) rank[sa[i]] = i;
        int[] lcp = new int[n];
        int h = 0;
        for (int i = 0; i < n; i++) {
            if (rank[i] > 0) {
                int j = sa[rank[i] - 1];
                while (i + h < n && j + h < n && s.charAt(i + h) == s.charAt(j + h)) h++;
                lcp[rank[i]] = h;
                if (h > 0) h--;
            } else {
                h = 0;
            }
        }
        return lcp;
    }
}
```

### Example

```
Distinct substrings of 'aaa': 3
Distinct substrings of 'abc': 6
```

`"aaa"` has only `3` distinct substrings (`"a"`, `"aa"`, `"aaa"`) despite `3(3+1)/2 = 6` total substring occurrences — the other `3` are duplicates, exactly matching the LCP array's sum. `"abc"` has no repeated characters at all, so every one of its `6` substrings is distinct.

## 4. Aho-Corasick Multi-Pattern Matching

### Theory

**Classic example — Aho-Corasick Automaton:** given **multiple** patterns, find every occurrence of every pattern in a text, in a **single pass** — the multi-pattern generalization of [Lesson 9's KMP](<./09. Pattern Matching.md#3-knuth-morris-pratt-kmp>). Build a **trie** of all patterns first; then, just like KMP's failure function tells you "how far back to fall on a mismatch," Aho-Corasick computes a **fail link** for every trie node (via BFS) pointing to the longest proper suffix of that node's path that is also a prefix of *some* pattern. While scanning the text, a mismatch at any depth falls back along fail links instead of restarting from the trie root — and every fail link also merges in whatever pattern matches were already found along the shorter path, so patterns that are suffixes of other patterns (like `"he"` inside `"she"`) are never missed.

### Code

```java
import java.util.ArrayDeque;
import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;
import java.util.Map;
import java.util.Queue;

public class AhoCorasick {
    static class Node {
        Map<Character, Node> children = new HashMap<>();
        Node fail;
        List<String> output = new ArrayList<>();
    }

    private final Node root = new Node();

    public static void main(String[] args) {
        AhoCorasick ac = new AhoCorasick();
        for (String pattern : new String[] { "he", "she", "his", "hers" }) {
            ac.insert(pattern);
        }
        ac.buildFailLinks();
        ac.search("ahishers");
    }

    void insert(String word) {
        Node node = root;
        for (char c : word.toCharArray()) {
            node = node.children.computeIfAbsent(c, k -> new Node());
        }
        node.output.add(word);
    }

    void buildFailLinks() {
        Queue<Node> queue = new ArrayDeque<>();
        for (Node child : root.children.values()) {
            child.fail = root;
            queue.add(child);
        }

        while (!queue.isEmpty()) {
            Node current = queue.poll();
            for (Map.Entry<Character, Node> entry : current.children.entrySet()) {
                char c = entry.getKey();
                Node child = entry.getValue();
                queue.add(child);

                Node fallback = current.fail;
                while (fallback != null && !fallback.children.containsKey(c)) {
                    fallback = fallback.fail;
                }
                child.fail = (fallback != null) ? fallback.children.get(c) : root;
                if (child.fail == null || child.fail == child) child.fail = root;
                child.output.addAll(child.fail.output);
            }
        }
    }

    void search(String text) {
        Node node = root;
        for (int i = 0; i < text.length(); i++) {
            char c = text.charAt(i);
            while (node != root && !node.children.containsKey(c)) {
                node = node.fail;
            }
            node = node.children.getOrDefault(c, root);

            for (String pattern : node.output) {
                int startIndex = i - pattern.length() + 1;
                System.out.println("Match '" + pattern + "' at index " + startIndex);
            }
        }
    }
}
```

### Example

```
Match 'his' at index 1
Match 'she' at index 3
Match 'he' at index 4
Match 'hers' at index 4
```

Scanning `"ahishers"` once finds all four patterns' occurrences, including the **overlapping** matches `"he"` and `"hers"` both starting at index 4, and `"she"` and `"he"` sharing characters — no re-scanning of the text per pattern was needed.
