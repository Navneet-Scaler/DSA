## 1. Longest Common Prefix

### Theory

**Classic example — Longest Common Prefix:** given an array of strings, find the longest prefix common to **all** of them. Simplest approach — **horizontal scanning**: start with the first string as the candidate prefix, then repeatedly shrink it (chop off the last character) until every remaining string actually starts with it. If the prefix shrinks to empty, there's no common prefix at all.

### Code

```java
public class LongestCommonPrefix {
    public static void main(String[] args) {
        System.out.println("LCP: " + longestCommonPrefix(new String[] { "flower", "flow", "flight" }));
        System.out.println("LCP: " + longestCommonPrefix(new String[] { "dog", "racecar", "car" }));
    }

    static String longestCommonPrefix(String[] strs) {
        if (strs.length == 0) return "";
        String prefix = strs[0];

        for (int i = 1; i < strs.length; i++) {
            while (!strs[i].startsWith(prefix)) {
                prefix = prefix.substring(0, prefix.length() - 1);
                if (prefix.isEmpty()) return "";
            }
        }
        return prefix;
    }
}
```

### Example

```
LCP: fl
LCP: 
```

`"flower"`, `"flow"`, `"flight"` all start with `"fl"`. `"dog"`, `"racecar"`, `"car"` share no starting character at all — empty string.

> **Java idiom:** `strs[i].startsWith(prefix)` is the built-in, readable way to express this check — no manual character-by-character comparison needed.

## 2. Longest Common Prefix via Binary Search

### Theory

Same problem, different technique: **binary search on the answer's length**. The set of lengths for which "all strings share a common prefix of this length" is monotonic — if length `k` works, every length `< k` also works — so binary search the largest working length between `0` and the shortest string's length, testing each candidate length by checking whether all strings share that exact prefix. Same **O(S)** total character comparisons in the worst case as horizontal scanning, but a useful alternative technique to recognize, especially when the "does length k work" check can itself be made cheap or parallelizable.

### Code

```java
public class LongestCommonPrefixBinarySearch {
    public static void main(String[] args) {
        System.out.println("LCP (binary search): "
                + longestCommonPrefix(new String[] { "flower", "flow", "flight" }));
    }

    static String longestCommonPrefix(String[] strs) {
        if (strs.length == 0) return "";
        int minLen = Integer.MAX_VALUE;
        for (String s : strs) minLen = Math.min(minLen, s.length());

        int lo = 0, hi = minLen;
        while (lo < hi) {
            int mid = (lo + hi + 1) / 2;
            if (isCommonPrefix(strs, mid)) {
                lo = mid;
            } else {
                hi = mid - 1;
            }
        }
        return strs[0].substring(0, lo);
    }

    static boolean isCommonPrefix(String[] strs, int length) {
        String prefix = strs[0].substring(0, length);
        for (String s : strs) {
            if (!s.startsWith(prefix)) return false;
        }
        return true;
    }
}
```

### Example

```
LCP (binary search): fl
```

Same result as horizontal scanning, reached by narrowing the candidate length `[0, minLength]` instead of shrinking a starting guess character by character.

## 3. Longest Common Suffix

### Theory

**Mirror of [Longest Common Prefix](#1-longest-common-prefix):** find the longest suffix shared by every string. Rather than writing a whole new algorithm, **reverse every string**, find their longest common *prefix* using the exact same technique, then reverse that result back — turning a new problem into a direct reuse of an already-solved one.

### Code

```java
public class LongestCommonSuffix {
    public static void main(String[] args) {
        System.out.println("LCS (suffix): "
                + longestCommonSuffix(new String[] { "coding", "training", "singing" }));
    }

    static String longestCommonSuffix(String[] strs) {
        String[] reversed = new String[strs.length];
        for (int i = 0; i < strs.length; i++) {
            reversed[i] = new StringBuilder(strs[i]).reverse().toString();
        }

        String prefixOfReversed = longestCommonPrefix(reversed);
        return new StringBuilder(prefixOfReversed).reverse().toString();
    }

    static String longestCommonPrefix(String[] strs) {
        if (strs.length == 0) return "";
        String prefix = strs[0];
        for (int i = 1; i < strs.length; i++) {
            while (!strs[i].startsWith(prefix)) {
                prefix = prefix.substring(0, prefix.length() - 1);
                if (prefix.isEmpty()) return "";
            }
        }
        return prefix;
    }
}
```

### Example

```
LCS (suffix): ing
```

`"coding"`, `"training"`, `"singing"` all end in `"ing"` — reversed, they all start with `"gni"`, whose reverse is `"ing"`.

## 4. Longest Happy Prefix

### Theory

**Classic example — Longest Happy Prefix:** find the longest proper prefix of a **single** string that is also a suffix of that same string (a "happy prefix" — proper meaning it can't be the entire string). This is **exactly** the [KMP prefix function / LPS array](<./09. Pattern Matching.md#2-prefix-function-lps-array>) from Lesson 9 — the last value in the LPS array, `lps[n-1]`, already **is** the length of the longest proper prefix that's also a suffix, since that's precisely what the LPS array was built to track. The two topics converge: what looked like a "prefix vs. suffix, across one string" problem is solved by the same array used to speed up pattern matching across two strings.

### Code

```java
public class LongestHappyPrefix {
    public static void main(String[] args) {
        System.out.println("Longest happy prefix of 'level': " + longestPrefix("level"));
        System.out.println("Longest happy prefix of 'ababab': " + longestPrefix("ababab"));
        System.out.println("Longest happy prefix of 'leetcodeleet': " + longestPrefix("leetcodeleet"));
    }

    static String longestPrefix(String s) {
        int n = s.length();
        int[] lps = new int[n];
        int length = 0;
        int i = 1;

        while (i < n) {
            if (s.charAt(i) == s.charAt(length)) {
                length++;
                lps[i] = length;
                i++;
            } else if (length != 0) {
                length = lps[length - 1];
            } else {
                lps[i] = 0;
                i++;
            }
        }
        return s.substring(0, lps[n - 1]);
    }
}
```

### Example

```
Longest happy prefix of 'level': l
Longest happy prefix of 'ababab': abab
Longest happy prefix of 'leetcodeleet': leet
```

`"level"`: only single-character `"l"` matches at both ends. `"ababab"`: `"abab"` (length 4) appears as both prefix and suffix. `"leetcodeleet"`: `"leet"` opens and closes the string.

| Technique | Complexity | Problems covered |
|---|---|---|
| Longest Common Prefix | O(S) total characters | Longest Common Prefix (horizontal scan) |
