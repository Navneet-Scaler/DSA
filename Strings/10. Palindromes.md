# 10. Palindromes

A palindrome reads the same forward and backward — a simple property that generates a surprisingly deep family of problems, from a basic yes/no check up to Manacher's O(n) algorithm for finding the longest palindromic substring. This lesson works through that whole progression.

## 1. Palindrome Check

### Theory

The basic check: two pointers converge from both ends, comparing characters — full code (plus the "ignore case and punctuation" variant, **Valid Palindrome**) is in [Lesson 6: Palindrome Checking](<./06. Two Pointers.md#4-palindrome-checking>), which also covers the harder **Valid Palindrome II** (allow deleting one character).

### Code

```java
public class PalindromeCheck {
    public static void main(String[] args) {
        System.out.println("'racecar' is a palindrome: " + isPalindrome("racecar"));
        System.out.println("'hello' is a palindrome: " + isPalindrome("hello"));
    }

    static boolean isPalindrome(String s) {
        int left = 0, right = s.length() - 1;
        while (left < right) {
            if (s.charAt(left) != s.charAt(right)) return false;
            left++;
            right--;
        }
        return true;
    }
}
```

### Example

```
'racecar' is a palindrome: true
'hello' is a palindrome: false
```

## 2. Expand Around Center

### Theory

**Classic example — Longest Palindromic Substring.** Every palindrome has a center — either a single character (odd length) or a gap between two characters (even length). Try **every possible center** (`2n - 1` of them), expanding outward while characters match, and track the longest expansion found. **O(n²)** — for each of the `O(n)` centers, expansion can take `O(n)`.

### Code

```java
public class ExpandAroundCenter {
    public static void main(String[] args) {
        System.out.println("Longest palindrome in 'babad': " + longestPalindrome("babad"));
        System.out.println("Longest palindrome in 'cbbd': " + longestPalindrome("cbbd"));
    }

    static String longestPalindrome(String s) {
        if (s.isEmpty()) return "";
        int start = 0, maxLen = 1;

        for (int i = 0; i < s.length(); i++) {
            int len1 = expand(s, i, i);       // odd-length palindromes, centered on i
            int len2 = expand(s, i, i + 1);   // even-length palindromes, centered between i and i+1

            int longer = Math.max(len1, len2);
            if (longer > maxLen) {
                maxLen = longer;
                start = i - (longer - 1) / 2;
            }
        }
        return s.substring(start, start + maxLen);
    }

    static int expand(String s, int left, int right) {
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            left--;
            right++;
        }
        return right - left - 1;
    }
}
```

### Example

```
Longest palindrome in 'babad': bab
Longest palindrome in 'cbbd': bb
```

### Alternative Approach: Dynamic Programming Table

**Theory:** The same problem solved differently — `dp[i][j]` is `true` if `s[i..j]` is a palindrome, built from smaller palindromes outward: `s[i..j]` is a palindrome exactly when `s[i] == s[j]` **and** `s[i+1..j-1]` is also a palindrome (or is short enough to not need checking). Also **O(n²)**, but O(n²) space too (vs. expand-around-center's O(1)) — included to show the DP framing this problem is also commonly taught with, which becomes essential once problems need to **count** or **reuse** palindrome-validity checks across many substrings (as in [Palindrome Partitioning](#5-palindrome-partitioning)).

```java
public class LongestPalindromeDP {
    public static void main(String[] args) {
        System.out.println("Longest palindrome in 'babad': " + longestPalindromeDP("babad"));
    }

    static String longestPalindromeDP(String s) {
        int n = s.length();
        if (n == 0) return "";
        boolean[][] dp = new boolean[n][n];
        int start = 0, maxLen = 1;

        for (int i = 0; i < n; i++) dp[i][i] = true;

        for (int length = 2; length <= n; length++) {
            for (int i = 0; i + length - 1 < n; i++) {
                int j = i + length - 1;
                if (s.charAt(i) == s.charAt(j) && (length == 2 || dp[i + 1][j - 1])) {
                    dp[i][j] = true;
                    if (length > maxLen) {
                        start = i;
                        maxLen = length;
                    }
                }
            }
        }
        return s.substring(start, start + maxLen);
    }
}
```

```
Longest palindrome in 'babad': bab
```

## 3. Longest Palindrome

### Theory

**Classic example — Longest Palindrome (build, not substring):** given a bag of characters, find the length of the **longest palindrome you can build** by rearranging them (not requiring them to be contiguous in the original string). Every character with an **even** count contributes its full count (split evenly on both sides of the palindrome); characters with an **odd** count contribute one less than their count (the leftover `1` can only be used once, as a single central character).

```java
import java.util.HashMap;
import java.util.Map;

public class LongestPalindromeBuild {
    public static void main(String[] args) {
        System.out.println("Longest buildable palindrome from 'abccccdd': " + longestPalindrome("abccccdd"));
    }

    static int longestPalindrome(String s) {
        Map<Character, Integer> freq = new HashMap<>();
        for (char c : s.toCharArray()) freq.merge(c, 1, Integer::sum);

        int length = 0;
        boolean hasOdd = false;
        for (int count : freq.values()) {
            length += (count / 2) * 2;
            if (count % 2 == 1) hasOdd = true;
        }
        return length + (hasOdd ? 1 : 0);
    }
}
```

### Example

```
Longest buildable palindrome from 'abccccdd': 7
```

`"abccccdd"`: `c` (count 4) and `d` (count 2) contribute fully; `a` and `b` (count 1 each) can only contribute one of them as the single center character — `4 + 2 + 1 = 7`.

## 4. Count Palindromes

### Theory

**Classic example — Count Palindromic Substrings:** count **every** palindromic substring (not just find the longest). Same [expand-around-center](#2-expand-around-center) idea, but instead of tracking the longest, increment a counter on every successful expansion step (each step found one more valid palindrome).

### Code

```java
public class CountPalindromicSubstrings {
    public static void main(String[] args) {
        System.out.println("Palindromic substrings in 'abc': " + countSubstrings("abc"));
        System.out.println("Palindromic substrings in 'aaa': " + countSubstrings("aaa"));
    }

    static int countSubstrings(String s) {
        int count = 0;
        for (int i = 0; i < s.length(); i++) {
            count += expandAndCount(s, i, i);
            count += expandAndCount(s, i, i + 1);
        }
        return count;
    }

    static int expandAndCount(String s, int left, int right) {
        int count = 0;
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            count++;
            left--;
            right++;
        }
        return count;
    }
}
```

### Example

```
Palindromic substrings in 'abc': 3
Palindromic substrings in 'aaa': 6
```

`"abc"` has 3 (each single character). `"aaa"` has 6: `"a","a","a","aa","aa","aaa"` — every substring of an all-same-character string is a palindrome.

## 5. Palindrome Partitioning

### Theory

**Classic example — Palindrome Partitioning:** split a string into the **fewest or all possible** ways such that every piece is itself a palindrome. The all-partitions version uses [backtracking](<./18. Backtracking.md>): at each position, try every prefix that's a palindrome, recurse on the remainder, and backtrack. (The **minimum cuts** variant, Palindrome Partitioning II, needs dynamic programming instead of full enumeration — covered in [Lesson 17: DP on Strings](<./17. Dynamic Programming on Strings.md>).)

### Code

```java
import java.util.ArrayList;
import java.util.List;

public class PalindromePartitioning {
    public static void main(String[] args) {
        System.out.println("All palindrome partitions of 'aab': " + partition("aab"));
    }

    static List<List<String>> partition(String s) {
        List<List<String>> result = new ArrayList<>();
        backtrack(s, 0, new ArrayList<>(), result);
        return result;
    }

    static void backtrack(String s, int start, List<String> current, List<List<String>> result) {
        if (start == s.length()) {
            result.add(new ArrayList<>(current));
            return;
        }
        for (int end = start + 1; end <= s.length(); end++) {
            String piece = s.substring(start, end);
            if (isPalindrome(piece)) {
                current.add(piece);
                backtrack(s, end, current, result);
                current.remove(current.size() - 1);
            }
        }
    }

    static boolean isPalindrome(String s) {
        int left = 0, right = s.length() - 1;
        while (left < right) {
            if (s.charAt(left) != s.charAt(right)) return false;
            left++;
            right--;
        }
        return true;
    }
}
```

### Example

```
All palindrome partitions of 'aab': [[a, a, b], [aa, b]]
```

## 6. Manacher's Algorithm

### Theory

[Expand Around Center](#2-expand-around-center) is O(n²) because expansion at each center can, in the worst case, re-examine characters already covered by a previous center's expansion. **Manacher's Algorithm** avoids this entirely, achieving **O(n)**: transform the string by inserting a separator (`#`) between every character (unifying odd- and even-length cases into one), then compute a "radius array" using a technique similar to the [Z-algorithm](<./09. Pattern Matching.md#5-z-algorithm>) — maintaining the rightmost-known palindrome boundary so far, and reusing previously-computed radius information via **mirror symmetry** whenever the current position falls inside that boundary.

### Code

```java
public class ManachersAlgorithm {
    public static void main(String[] args) {
        System.out.println("Longest palindrome (Manacher's), 'babad': " + longestPalindrome("babad"));
        System.out.println("Longest palindrome (Manacher's), 'cbbd': " + longestPalindrome("cbbd"));
    }

    static String longestPalindrome(String s) {
        String t = transform(s);
        int n = t.length();
        int[] radius = new int[n];
        int center = 0, rightBoundary = 0;

        for (int i = 0; i < n; i++) {
            if (i < rightBoundary) {
                radius[i] = Math.min(rightBoundary - i, radius[2 * center - i]);
            }
            while (i - radius[i] - 1 >= 0 && i + radius[i] + 1 < n
                    && t.charAt(i - radius[i] - 1) == t.charAt(i + radius[i] + 1)) {
                radius[i]++;
            }
            if (i + radius[i] > rightBoundary) {
                center = i;
                rightBoundary = i + radius[i];
            }
        }

        int maxLen = 0, centerIndex = 0;
        for (int i = 0; i < n; i++) {
            if (radius[i] > maxLen) {
                maxLen = radius[i];
                centerIndex = i;
            }
        }
        int start = (centerIndex - maxLen) / 2;
        return s.substring(start, start + maxLen);
    }

    static String transform(String s) {
        StringBuilder sb = new StringBuilder("#");
        for (char c : s.toCharArray()) {
            sb.append(c).append('#');
        }
        return sb.toString();
    }
}
```

### Example

```
Longest palindrome (Manacher's), 'babad': bab
Longest palindrome (Manacher's), 'cbbd': bb
```

Same answers as [Expand Around Center](#2-expand-around-center) and the DP table version — three different techniques converging on the same result, at three different complexity/space trade-offs.
