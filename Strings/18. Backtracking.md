# 18. Backtracking

Backtracking builds a solution incrementally, one choice at a time, and **undoes** a choice ("backtracks") as soon as it's clear it can't lead anywhere valid. String problems are a natural fit — build up a string or a set of segments character-by-character, pruning branches that can already be seen to violate a constraint (parenthesis balance, valid IP octet ranges, cell already visited). [Lesson 10's Palindrome Partitioning](<./10. Palindromes.md#5-palindrome-partitioning>) was this lesson's first example; this one covers the rest of the classic backtracking-on-strings problem set.

## 1. Generate Parentheses

### Theory

**Classic example — Generate Parentheses:** generate all valid combinations of `n` pairs of balanced parentheses. Track how many `(` and `)` have been placed so far: add `(` whenever fewer than `n` have been used, and add `)` only when doing so wouldn't create more closing than opening brackets (`closeCount < openCount`) — this constraint alone guarantees every generated string is valid, so no separate validity check is needed at the end.

### Code

```java
import java.util.ArrayList;
import java.util.List;

public class GenerateParentheses {
    public static void main(String[] args) {
        System.out.println("generateParenthesis(3): " + generateParenthesis(3));
    }

    static List<String> generateParenthesis(int n) {
        List<String> result = new ArrayList<>();
        backtrack(result, new StringBuilder(), 0, 0, n);
        return result;
    }

    static void backtrack(List<String> result, StringBuilder current, int openCount, int closeCount, int n) {
        if (current.length() == 2 * n) {
            result.add(current.toString());
            return;
        }
        if (openCount < n) {
            current.append('(');
            backtrack(result, current, openCount + 1, closeCount, n);
            current.deleteCharAt(current.length() - 1);
        }
        if (closeCount < openCount) {
            current.append(')');
            backtrack(result, current, openCount, closeCount + 1, n);
            current.deleteCharAt(current.length() - 1);
        }
    }
}
```

### Example

```
generateParenthesis(3): [((())), (()()), (())(), ()(()), ()()()]
```

All `5` valid ways to balance `3` pairs of parentheses — the Catalan number `C(3) = 5`.

## 2. Letter Combinations of a Phone Number

### Theory

**Classic example — Letter Combinations of a Phone Number:** given a string of digits `2`-`9`, return every possible letter combination the digits could represent (like an old T9 keypad). At each recursive step, pick **one** letter from the current digit's set and recurse on the remaining digits — every complete assignment (one letter chosen per digit) becomes one output combination.

### Code

```java
import java.util.ArrayList;
import java.util.List;
import java.util.Map;

public class LetterCombinations {
    public static void main(String[] args) {
        System.out.println("letterCombinations('23'): " + letterCombinations("23"));
    }

    static final Map<Character, String> PHONE = Map.of(
            '2', "abc", '3', "def", '4', "ghi", '5', "jkl",
            '6', "mno", '7', "pqrs", '8', "tuv", '9', "wxyz");

    static List<String> letterCombinations(String digits) {
        List<String> result = new ArrayList<>();
        if (digits.isEmpty()) return result;
        backtrack(result, new StringBuilder(), digits, 0);
        return result;
    }

    static void backtrack(List<String> result, StringBuilder current, String digits, int index) {
        if (index == digits.length()) {
            result.add(current.toString());
            return;
        }
        String letters = PHONE.get(digits.charAt(index));
        for (char c : letters.toCharArray()) {
            current.append(c);
            backtrack(result, current, digits, index + 1);
            current.deleteCharAt(current.length() - 1);
        }
    }
}
```

### Example

```
letterCombinations('23'): [ad, ae, af, bd, be, bf, cd, ce, cf]
```

`2` → `{a,b,c}`, `3` → `{d,e,f}` — every one of the `3 × 3 = 9` pairings appears exactly once.

## 3. Permutations of a String

### Theory

**Classic example — Permutations of a String (with duplicate characters):** generate all **distinct** permutations of a string, even when it has repeated characters (a plain "try every unused character" backtrack would generate the same permutation multiple times). The fix: **sort** the characters first, then at each recursive position, skip a character if it's the same as the previous one **and** the previous one hasn't been used yet in this branch — this specific condition (`!used[i-1]`) ensures duplicates are only tried in a fixed left-to-right order, eliminating duplicate output without eliminating valid distinct arrangements.

### Code

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

public class PermutationsOfString {
    public static void main(String[] args) {
        System.out.println("permuteUnique('abc'): " + permuteUnique("abc"));
        System.out.println("permuteUnique('aab'): " + permuteUnique("aab"));
    }

    static List<String> permuteUnique(String s) {
        char[] chars = s.toCharArray();
        Arrays.sort(chars);
        List<String> result = new ArrayList<>();
        boolean[] used = new boolean[chars.length];
        backtrack(result, new StringBuilder(), chars, used);
        return result;
    }

    static void backtrack(List<String> result, StringBuilder current, char[] chars, boolean[] used) {
        if (current.length() == chars.length) {
            result.add(current.toString());
            return;
        }
        for (int i = 0; i < chars.length; i++) {
            if (used[i]) continue;
            if (i > 0 && chars[i] == chars[i - 1] && !used[i - 1]) continue; // skip duplicate branch
            used[i] = true;
            current.append(chars[i]);
            backtrack(result, current, chars, used);
            current.deleteCharAt(current.length() - 1);
            used[i] = false;
        }
    }
}
```

### Example

```
permuteUnique('abc'): [abc, acb, bac, bca, cab, cba]
permuteUnique('aab'): [aab, aba, baa]
```

`"abc"` (no duplicates) produces all `3! = 6` permutations. `"aab"` produces only `3` **distinct** permutations, not `3! = 6` — the duplicate-skip condition prevents the two `a`s from producing indistinguishable repeats.

## 4. Restore IP Addresses

### Theory

**Classic example — Restore IP Addresses:** given a string of digits, insert dots to form every valid IPv4 address (four segments, each `0`–`255`, with no leading zeros unless the segment is exactly `"0"`). Backtrack by trying every valid segment length (`1`, `2`, or `3` digits) at each step, validating the segment (range check + leading-zero check) before recursing — once **4 segments** have been placed, the attempt succeeds only if it also **exactly consumed** the whole input string.

### Code

```java
import java.util.ArrayList;
import java.util.List;

public class RestoreIpAddresses {
    public static void main(String[] args) {
        System.out.println("restoreIpAddresses('25525511135'): " + restoreIpAddresses("25525511135"));
    }

    static List<String> restoreIpAddresses(String s) {
        List<String> result = new ArrayList<>();
        backtrack(result, new ArrayList<>(), s, 0);
        return result;
    }

    static void backtrack(List<String> result, List<String> segments, String s, int start) {
        if (segments.size() == 4) {
            if (start == s.length()) {
                result.add(String.join(".", segments));
            }
            return;
        }
        for (int length = 1; length <= 3 && start + length <= s.length(); length++) {
            String segment = s.substring(start, start + length);
            if (isValidSegment(segment)) {
                segments.add(segment);
                backtrack(result, segments, s, start + length);
                segments.remove(segments.size() - 1);
            }
        }
    }

    static boolean isValidSegment(String segment) {
        if (segment.length() > 1 && segment.charAt(0) == '0') return false;
        int value = Integer.parseInt(segment);
        return value >= 0 && value <= 255;
    }
}
```

### Example

```
restoreIpAddresses('25525511135'): [255.255.11.135, 255.255.111.35]
```

Only these `2` of the many possible dot-placements form valid IPv4 addresses — every other split either produces a segment `> 255` or one with an invalid leading zero.

## 5. Word Search

### Theory

**Classic example — Word Search (advanced):** given a 2D grid of letters, determine if a word can be formed by tracing a path through **adjacent** cells (up/down/left/right), never reusing the same cell twice within one path. Backtrack from every starting cell: at each step, check the current cell matches the next needed character, **temporarily mark it visited** (overwrite with a sentinel like `'#'`), recurse into all 4 neighbors looking for the next character, then **restore the original character** on the way back out — this in-place marking avoids needing a separate `visited` grid.

### Code

```java
public class WordSearch {
    public static void main(String[] args) {
        char[][] board = {
                { 'A', 'B', 'C', 'E' },
                { 'S', 'F', 'C', 'S' },
                { 'A', 'D', 'E', 'E' }
        };
        System.out.println("exist(board,'ABCCED'): " + exist(board, "ABCCED"));
        System.out.println("exist(board,'SEE'): " + exist(board, "SEE"));
        System.out.println("exist(board,'ABCB'): " + exist(board, "ABCB"));
    }

    static boolean exist(char[][] board, String word) {
        int rows = board.length, cols = board[0].length;
        for (int r = 0; r < rows; r++) {
            for (int c = 0; c < cols; c++) {
                if (backtrack(board, word, r, c, 0)) return true;
            }
        }
        return false;
    }

    static boolean backtrack(char[][] board, String word, int r, int c, int index) {
        if (index == word.length()) return true;
        if (r < 0 || r >= board.length || c < 0 || c >= board[0].length
                || board[r][c] != word.charAt(index)) {
            return false;
        }

        char temp = board[r][c];
        board[r][c] = '#'; // mark visited in place

        boolean found = backtrack(board, word, r + 1, c, index + 1)
                || backtrack(board, word, r - 1, c, index + 1)
                || backtrack(board, word, r, c + 1, index + 1)
                || backtrack(board, word, r, c - 1, index + 1);

        board[r][c] = temp; // restore on the way back out
        return found;
    }
}
```

### Example

```
exist(board,'ABCCED'): true
exist(board,'SEE'): true
exist(board,'ABCB'): false
```

`"ABCCED"` traces a valid connected path through the grid. `"ABCB"` fails — after `A→B→C`, the only remaining `B` isn't adjacent to that `C`, and no other path reaches a second `B` without reusing a cell.
