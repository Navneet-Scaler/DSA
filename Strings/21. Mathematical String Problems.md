## 1. Excel Column Conversion

### Theory

**Classic example — Excel Sheet Column Title / Column Number:** spreadsheet columns are labeled `A, B, ..., Z, AA, AB, ...` — a base-26 numbering system, but with a twist: there's **no digit for zero** (there's no "column 0"), so it's not quite standard base-26. The fix for **number → title**: before taking `n % 26`, **decrement `n` by 1** first — this shifts the range from `1-26` down to `0-25`, letting `0` map to `'A'` just like a normal base-26 digit would. For **title → number**, the reverse is direct: accumulate `result = result * 26 + (digit value, 1-indexed)` reading left to right.

### Code

```java
public class ExcelColumnConversion {
    public static void main(String[] args) {
        System.out.println("convertToTitle(1): " + convertToTitle(1));
        System.out.println("convertToTitle(28): " + convertToTitle(28));
        System.out.println("convertToTitle(701): " + convertToTitle(701));

        System.out.println("titleToNumber('A'): " + titleToNumber("A"));
        System.out.println("titleToNumber('AB'): " + titleToNumber("AB"));
        System.out.println("titleToNumber('ZY'): " + titleToNumber("ZY"));
    }

    static String convertToTitle(int n) {
        StringBuilder result = new StringBuilder();
        while (n > 0) {
            n--; // shift into 0-25 range -- there's no "digit zero" column
            result.append((char) ('A' + n % 26));
            n /= 26;
        }
        return result.reverse().toString();
    }

    static int titleToNumber(String s) {
        int result = 0;
        for (char c : s.toCharArray()) {
            result = result * 26 + (c - 'A' + 1);
        }
        return result;
    }
}
```

### Example

```
convertToTitle(1): A
convertToTitle(28): AB
convertToTitle(701): ZY
titleToNumber('A'): 1
titleToNumber('AB'): 28
titleToNumber('ZY'): 701
```

`28` → `"AB"`: without the `n--` adjustment, `28 % 26 = 2` would give `'C'` instead of the correct `'B'` — the decrement is what makes the "no zero digit" base-26 system work.

## 2. Multiply Strings

### Theory

**Classic example — Multiply Strings:** multiply two non-negative integers represented as strings, without converting them to `int`/`long` (they may be arbitrarily large). Simulate elementary-school long multiplication: multiplying digit `i` of `num1` by digit `j` of `num2` contributes to positions `i+j` and `i+j+1` of the result array (since two single digits multiply to at most two digits). Accumulate all these partial products into a fixed-size result array first, **then** resolve carries in a single final pass — this deferred-carry approach is simpler than carrying immediately after every single-digit multiplication.

### Code

```java
public class MultiplyStrings {
    public static void main(String[] args) {
        System.out.println("multiply('123','456'): " + multiply("123", "456"));
        System.out.println("multiply('2','3'): " + multiply("2", "3"));
    }

    static String multiply(String num1, String num2) {
        if (num1.equals("0") || num2.equals("0")) return "0";

        int m = num1.length(), n = num2.length();
        int[] result = new int[m + n];

        for (int i = m - 1; i >= 0; i--) {
            for (int j = n - 1; j >= 0; j--) {
                int mul = (num1.charAt(i) - '0') * (num2.charAt(j) - '0');
                int p1 = i + j, p2 = i + j + 1;
                int total = mul + result[p2];

                result[p2] = total % 10;
                result[p1] += total / 10;
            }
        }

        StringBuilder sb = new StringBuilder();
        for (int digit : result) sb.append(digit);

        int firstNonZero = 0;
        while (firstNonZero < sb.length() - 1 && sb.charAt(firstNonZero) == '0') firstNonZero++;
        return sb.substring(firstNonZero);
    }
}
```

### Example

```
multiply('123','456'): 56088
multiply('2','3'): 6
```

`123 × 456 = 56088` — computed purely through digit-by-digit multiplication and carry propagation, matching what `Long.parseLong` arithmetic would give, but works for numbers too large to fit in any primitive type.

## 3. Fraction to Recurring Decimal

### Theory

**Classic example — Fraction to Recurring Decimal:** convert a fraction (numerator/denominator) to its decimal string representation, wrapping any **repeating** portion in parentheses (e.g. `4/333 = 0.(012)`). Long division naturally produces digits one at a time from the remainder; the key insight is that a remainder **must eventually repeat** (there are only `denominator` possible remainder values), and the moment a remainder repeats, the digit sequence from that point on will repeat too. Track a map from **remainder → position in the output** where it first appeared; when a remainder recurs, insert `(` at its first-seen position and close with `)` at the end.

### Code

```java
import java.util.HashMap;
import java.util.Map;

public class FractionToRecurringDecimal {
    public static void main(String[] args) {
        System.out.println("fractionToDecimal(1,2): " + fractionToDecimal(1, 2));
        System.out.println("fractionToDecimal(2,1): " + fractionToDecimal(2, 1));
        System.out.println("fractionToDecimal(4,333): " + fractionToDecimal(4, 333));
    }

    static String fractionToDecimal(long numerator, long denominator) {
        if (numerator == 0) return "0";

        StringBuilder result = new StringBuilder();
        if ((numerator < 0) != (denominator < 0)) result.append('-');
        numerator = Math.abs(numerator);
        denominator = Math.abs(denominator);

        result.append(numerator / denominator);
        long remainder = numerator % denominator;
        if (remainder == 0) return result.toString();

        result.append('.');
        Map<Long, Integer> seenAtPosition = new HashMap<>();

        while (remainder != 0) {
            if (seenAtPosition.containsKey(remainder)) {
                int insertAt = seenAtPosition.get(remainder);
                result.insert(insertAt, '(');
                result.append(')');
                break;
            }
            seenAtPosition.put(remainder, result.length());
            remainder *= 10;
            result.append(remainder / denominator);
            remainder %= denominator;
        }
        return result.toString();
    }
}
```

### Example

```
fractionToDecimal(1,2): 0.5
fractionToDecimal(2,1): 2
fractionToDecimal(4,333): 0.(012)
```

`4/333`: the long-division remainders cycle back to their starting value after producing `"012"`, so that block repeats forever — `"0.(012)"` captures it exactly, without printing an infinite string.

## 4. Integer to English Words

### Theory

**Classic example — Integer to English Words (advanced):** convert a number up to `2³¹ - 1` into its full English-word representation (e.g. `1234567` → `"One Million Two Hundred Thirty Four Thousand Five Hundred Sixty Seven"`). Break the number into **groups of three digits** (thousands, millions, billions), converting each group of up to 999 with a shared `helper` that handles hundreds, tens, and the irregular teens (`11`–`19` don't follow the "tens + ones" pattern, so they get their own lookup table). Process groups from the **largest** down, appending the appropriate scale word (`"Thousand"`, `"Million"`, `"Billion"`) after each non-zero group.

### Code

```java
public class IntegerToEnglishWords {
    static final String[] BELOW_20 = {
            "", "One", "Two", "Three", "Four", "Five", "Six", "Seven", "Eight", "Nine", "Ten",
            "Eleven", "Twelve", "Thirteen", "Fourteen", "Fifteen", "Sixteen", "Seventeen", "Eighteen", "Nineteen"
    };
    static final String[] TENS = {
            "", "", "Twenty", "Thirty", "Forty", "Fifty", "Sixty", "Seventy", "Eighty", "Ninety"
    };
    static final String[] THOUSANDS = { "", "Thousand", "Million", "Billion" };

    public static void main(String[] args) {
        System.out.println("numberToWords(123): " + numberToWords(123));
        System.out.println("numberToWords(12345): " + numberToWords(12345));
        System.out.println("numberToWords(1234567): " + numberToWords(1234567));
    }

    static String numberToWords(int num) {
        if (num == 0) return "Zero";

        StringBuilder result = new StringBuilder();
        int groupIndex = 0;

        while (num > 0) {
            if (num % 1000 != 0) {
                String group = helper(num % 1000) + THOUSANDS[groupIndex] + " ";
                result.insert(0, group);
            }
            num /= 1000;
            groupIndex++;
        }
        return result.toString().trim();
    }

    static String helper(int num) {
        if (num == 0) {
            return "";
        } else if (num < 20) {
            return BELOW_20[num] + " ";
        } else if (num < 100) {
            return TENS[num / 10] + " " + helper(num % 10);
        } else {
            return BELOW_20[num / 100] + " Hundred " + helper(num % 100);
        }
    }
}
```

### Example

```
numberToWords(123): One Hundred Twenty Three
numberToWords(12345): Twelve Thousand Three Hundred Forty Five
numberToWords(1234567): One Million Two Hundred Thirty Four Thousand Five Hundred Sixty Seven
```

`1234567` splits into groups `1` (billions-free, this is the millions group), `234` (thousands group), `567` (ones group) — each converted independently by `helper`, then joined with the correct scale word per group.
