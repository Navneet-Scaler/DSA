## 1. Edit Distance

### Theory

**Classic example — Edit Distance (Levenshtein Distance):** find the minimum number of insertions, deletions, and substitutions to turn `word1` into `word2`. `dp[i][j]` = edit distance between the first `i` characters of `word1` and first `j` of `word2`. If the current characters match, no edit is needed there (`dp[i-1][j-1]`); otherwise, take the best of three options plus one edit: **substitute** (`dp[i-1][j-1]`), **delete** from `word1` (`dp[i-1][j]`), or **insert** into `word1` (`dp[i][j-1]`).

### Code

```java
public class EditDistance {
    public static void main(String[] args) {
        System.out.println("Edit distance 'horse'->'ros': " + minDistance("horse", "ros"));
        System.out.println("Edit distance 'intention'->'execution': " + minDistance("intention", "execution"));
    }

    static int minDistance(String word1, String word2) {
        int m = word1.length(), n = word2.length();
        int[][] dp = new int[m + 1][n + 1];

        for (int i = 0; i <= m; i++) dp[i][0] = i;
        for (int j = 0; j <= n; j++) dp[0][j] = j;

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1];
                } else {
                    dp[i][j] = 1 + Math.min(dp[i - 1][j - 1], Math.min(dp[i - 1][j], dp[i][j - 1]));
                }
            }
        }
        return dp[m][n];
    }
}
```

### Example

```
Edit distance 'horse'->'ros': 3
Edit distance 'intention'->'execution': 5
```

`"horse"` → `"ros"`: delete `h`, replace `r→r` (match), replace `s→o`... concretely: `horse → rorse → rose → ros` — `3` edits.

## 2. Longest Common Substring

### Theory

**Contrasted with [Lesson 12's Longest Common Subsequence](<./12. Subsequences.md#2-longest-common-subsequence>):** a **substring** must be **contiguous**, unlike a subsequence. This changes the DP recurrence in one crucial way — `dp[i][j]` here means "length of the common substring **ending exactly** at `s1[i-1]` and `s2[j-1]`," not "best common subsequence using prefixes." When characters match, extend the diagonal (`dp[i-1][j-1] + 1`); when they don't, the streak is **broken entirely** — reset to `0` (LCS's subsequence version instead takes `max(dp[i-1][j], dp[i][j-1])`, since skipping is always allowed for subsequences but never for contiguous substrings). Track the running maximum and its ending position across the whole table, since (unlike LCS) the answer isn't guaranteed to be in the bottom-right cell.

### Code

```java
public class LongestCommonSubstring {
    public static void main(String[] args) {
        System.out.println("LCSubstring 'abcde'/'abfce': " + longestCommonSubstring("abcde", "abfce"));
        System.out.println("LCSubstring 'GeeksforGeeks'/'GeeksQuiz': "
                + longestCommonSubstring("GeeksforGeeks", "GeeksQuiz"));
    }

    static String longestCommonSubstring(String s1, String s2) {
        int m = s1.length(), n = s2.length();
        int[][] dp = new int[m + 1][n + 1];
        int maxLen = 0, endIndex = 0;

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (s1.charAt(i - 1) == s2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                    if (dp[i][j] > maxLen) {
                        maxLen = dp[i][j];
                        endIndex = i;
                    }
                }
                // else: dp[i][j] stays 0 -- the streak breaks entirely (unlike LCS)
            }
        }
        return s1.substring(endIndex - maxLen, endIndex);
    }
}
```

### Example

```
LCSubstring 'abcde'/'abfce': ab
LCSubstring 'GeeksforGeeks'/'GeeksQuiz': Geeks
```

`"abcde"` and `"abfce"` share `"ab"` contiguously (length 2) — even though `"ce"` also appears in both, non-contiguously, only the contiguous run counts here (the subsequence version from Lesson 12 would find the longer `"abce"`, length 4).

## 3. Word Break

### Theory

**Classic example — Word Break:** given a string and a dictionary of words, determine if the string can be segmented into a space-separated sequence of dictionary words. `dp[i]` = true if `s[0..i-1]` can be fully segmented. For each `i`, try every earlier split point `j`: if `dp[j]` is true **and** `s[j..i-1]` is itself a dictionary word, then `dp[i]` is true.

### Code

```java
import java.util.HashSet;
import java.util.List;
import java.util.Set;

public class WordBreak {
    public static void main(String[] args) {
        System.out.println("wordBreak('leetcode', [leet, code]): "
                + wordBreak("leetcode", List.of("leet", "code")));
        System.out.println("wordBreak('catsandog', [cats, dog, sand, and, cat]): "
                + wordBreak("catsandog", List.of("cats", "dog", "sand", "and", "cat")));
    }

    static boolean wordBreak(String s, List<String> wordDict) {
        Set<String> words = new HashSet<>(wordDict);
        int n = s.length();
        boolean[] dp = new boolean[n + 1];
        dp[0] = true;

        for (int i = 1; i <= n; i++) {
            for (int j = 0; j < i; j++) {
                if (dp[j] && words.contains(s.substring(j, i))) {
                    dp[i] = true;
                    break;
                }
            }
        }
        return dp[n];
    }
}
```

### Example

```
wordBreak('leetcode', [leet, code]): true
wordBreak('catsandog', [cats, dog, sand, and, cat]): false
```

`"leetcode"` splits cleanly into `"leet" + "code"`. `"catsandog"` has no valid full segmentation — `"cats"+"and"+"og"` fails since `"og"` isn't in the dictionary, and no other split point works either.

## 4. Wildcard and Regex Matching

### Theory

**Classic example — Wildcard Matching:** match a string against a pattern where `?` matches any single character and `*` matches **any sequence** of characters (including empty). `dp[i][j]` = whether `s[0..i-1]` matches `p[0..j-1]`. A `*` offers two choices — match it as **empty** (`dp[i][j-1]`) or **consume one more character of `s`** while keeping the `*` available for more (`dp[i-1][j]`) — taking either if it leads to a match.

### Code

```java
public class WildcardMatching {
    public static void main(String[] args) {
        System.out.println("wildcardMatch('aa','a'): " + isMatch("aa", "a"));
        System.out.println("wildcardMatch('aa','*'): " + isMatch("aa", "*"));
        System.out.println("wildcardMatch('cb','?a'): " + isMatch("cb", "?a"));
    }

    static boolean isMatch(String s, String p) {
        int m = s.length(), n = p.length();
        boolean[][] dp = new boolean[m + 1][n + 1];
        dp[0][0] = true;

        for (int j = 1; j <= n; j++) {
            if (p.charAt(j - 1) == '*') dp[0][j] = dp[0][j - 1];
        }

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                char pc = p.charAt(j - 1);
                if (pc == '*') {
                    dp[i][j] = dp[i - 1][j] || dp[i][j - 1];
                } else if (pc == '?' || pc == s.charAt(i - 1)) {
                    dp[i][j] = dp[i - 1][j - 1];
                }
            }
        }
        return dp[m][n];
    }
}
```

### Example

```
wildcardMatch('aa','a'): false
wildcardMatch('aa','*'): true
wildcardMatch('cb','?a'): false
```

### Problem 2 (Advanced): Regular Expression Matching

**Theory:** A related but distinct pattern language — `.` matches any single character (same as `?` above), but `*` here means **zero or more of the preceding character** (not "any sequence," and it always pairs with the character before it). The recurrence changes accordingly: `p[j-1] == '*'` looks at the pair `p[j-2]p[j-1]`, offering the choice to use it **zero times** (`dp[i][j-2]`, skip the pair entirely) or, if `p[j-2]` matches the current `s` character, **one more time** (`dp[i-1][j]`, keep consuming `s` while keeping the `*` pair available).

```java
public class RegularExpressionMatching {
    public static void main(String[] args) {
        System.out.println("regexMatch('aa','a*'): " + isMatch("aa", "a*"));
        System.out.println("regexMatch('ab','.*'): " + isMatch("ab", ".*"));
        System.out.println("regexMatch('mississippi','mis*is*p*.'): " + isMatch("mississippi", "mis*is*p*."));
    }

    static boolean isMatch(String s, String p) {
        int m = s.length(), n = p.length();
        boolean[][] dp = new boolean[m + 1][n + 1];
        dp[0][0] = true;

        for (int j = 1; j <= n; j++) {
            if (p.charAt(j - 1) == '*') dp[0][j] = dp[0][j - 2];
        }

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                char pc = p.charAt(j - 1);
                if (pc == '*') {
                    boolean zeroOccurrences = dp[i][j - 2];
                    boolean oneMoreOccurrence = (p.charAt(j - 2) == '.' || p.charAt(j - 2) == s.charAt(i - 1))
                            && dp[i - 1][j];
                    dp[i][j] = zeroOccurrences || oneMoreOccurrence;
                } else if (pc == '.' || pc == s.charAt(i - 1)) {
                    dp[i][j] = dp[i - 1][j - 1];
                }
            }
        }
        return dp[m][n];
    }
}
```

```
regexMatch('aa','a*'): true
regexMatch('ab','.*'): true
regexMatch('mississippi','mis*is*p*.'): false
```

`"a*"` means "zero or more `a`s," matching `"aa"`. `"mis*is*p*."` fails against `"mississippi"` — after matching `"mis"` and expanding `s*` to cover `"iss"`, the remaining `p*.` (zero-or-more `p` then any one character) can't stretch to cover the rest (`"issippi"`).

## 5. Palindrome Partitioning II

### Theory

**Classic example — Palindrome Partitioning II**, the harder sibling of [Lesson 10's Palindrome Partitioning](<./10. Palindromes.md#5-palindrome-partitioning>): instead of enumerating **all** ways to split a string into palindromic pieces, find the **minimum number of cuts** needed. Precompute a palindrome-check table `isPal[i][j]` (same DP as [Lesson 10's DP-table approach](<./10. Palindromes.md#alternative-approach-dynamic-programming-table>)), then a second DP `cuts[i]` = minimum cuts needed for `s[0..i]`: if the whole prefix is itself a palindrome, `0` cuts are needed; otherwise, try every split point `j` where `s[j+1..i]` is a palindrome, taking the best `cuts[j] + 1`.

### Code

```java
public class PalindromePartitioningII {
    public static void main(String[] args) {
        System.out.println("minCut 'aab': " + minCut("aab"));
        System.out.println("minCut 'a': " + minCut("a"));
        System.out.println("minCut 'ab': " + minCut("ab"));
    }

    static int minCut(String s) {
        int n = s.length();
        boolean[][] isPal = new boolean[n][n];
        for (int i = 0; i < n; i++) isPal[i][i] = true;

        for (int length = 2; length <= n; length++) {
            for (int i = 0; i + length - 1 < n; i++) {
                int j = i + length - 1;
                if (s.charAt(i) == s.charAt(j) && (length == 2 || isPal[i + 1][j - 1])) {
                    isPal[i][j] = true;
                }
            }
        }

        int[] cuts = new int[n];
        for (int i = 0; i < n; i++) {
            if (isPal[0][i]) {
                cuts[i] = 0;
                continue;
            }
            cuts[i] = Integer.MAX_VALUE;
            for (int j = 0; j < i; j++) {
                if (isPal[j + 1][i] && cuts[j] + 1 < cuts[i]) {
                    cuts[i] = cuts[j] + 1;
                }
            }
        }
        return cuts[n - 1];
    }
}
```

### Example

```
minCut 'aab': 1
minCut 'a': 0
minCut 'ab': 1
```

`"aab"` needs just `1` cut: `"aa" | "b"` — both pieces are palindromes. A single character never needs a cut.

## 6. Interleaving String

### Theory

**Classic example — Interleaving String (advanced):** determine whether `s3` can be formed by interleaving `s1` and `s2` while preserving each one's internal character order. `dp[i][j]` = whether the first `i + j` characters of `s3` can be formed by interleaving the first `i` characters of `s1` and first `j` of `s2`. At each cell, the next character of `s3` must come from **either** the next unused character of `s1` (if it matches and `dp[i-1][j]` held) **or** the next unused character of `s2` (if it matches and `dp[i][j-1]` held).

### Code

```java
public class InterleavingString {
    public static void main(String[] args) {
        System.out.println("isInterleave('aabcc','dbbca','aadbbcbcac'): "
                + isInterleave("aabcc", "dbbca", "aadbbcbcac"));
        System.out.println("isInterleave('aabcc','dbbca','aadbbbaccc'): "
                + isInterleave("aabcc", "dbbca", "aadbbbaccc"));
    }

    static boolean isInterleave(String s1, String s2, String s3) {
        int m = s1.length(), n = s2.length();
        if (m + n != s3.length()) return false;

        boolean[][] dp = new boolean[m + 1][n + 1];
        dp[0][0] = true;

        for (int i = 1; i <= m; i++) {
            dp[i][0] = dp[i - 1][0] && s1.charAt(i - 1) == s3.charAt(i - 1);
        }
        for (int j = 1; j <= n; j++) {
            dp[0][j] = dp[0][j - 1] && s2.charAt(j - 1) == s3.charAt(j - 1);
        }

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                boolean fromS1 = dp[i - 1][j] && s1.charAt(i - 1) == s3.charAt(i + j - 1);
                boolean fromS2 = dp[i][j - 1] && s2.charAt(j - 1) == s3.charAt(i + j - 1);
                dp[i][j] = fromS1 || fromS2;
            }
        }
        return dp[m][n];
    }
}
```

### Example

```
isInterleave('aabcc','dbbca','aadbbcbcac'): true
isInterleave('aabcc','dbbca','aadbbbaccc'): false
```

`"aadbbcbcac"` correctly interleaves `"aabcc"` and `"dbbca"` character-by-character while preserving both strings' internal orders; `"aadbbbaccc"` has the same character multiset but no valid interleaving order exists.
