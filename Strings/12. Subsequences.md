# 12. Subsequences

A **subsequence** keeps characters in order but doesn't require them to be contiguous (unlike a substring). That relaxation is exactly what makes subsequence problems need dynamic programming instead of the sliding-window/two-pointer tricks that work for substrings — this lesson goes from a simple existence check up to counting and comparing subsequences between two strings.

## 1. Subsequence Check

### Theory

**Classic example — Is Subsequence:** check whether `s` can be obtained by deleting some characters from `t` (keeping the remaining ones in order). Two pointers, one per string: advance through `t`, and whenever the current character matches the pointer's target in `s`, advance that pointer too. `s` is a subsequence of `t` exactly when its pointer reaches the end.

### Code

```java
public class IsSubsequence {
    public static void main(String[] args) {
        System.out.println("'abc' subsequence of 'ahbgdc': " + isSubsequence("abc", "ahbgdc"));
        System.out.println("'axc' subsequence of 'ahbgdc': " + isSubsequence("axc", "ahbgdc"));
    }

    static boolean isSubsequence(String s, String t) {
        int i = 0;
        for (int j = 0; j < t.length() && i < s.length(); j++) {
            if (s.charAt(i) == t.charAt(j)) {
                i++;
            }
        }
        return i == s.length();
    }
}
```

### Example

```
'abc' subsequence of 'ahbgdc': true
'axc' subsequence of 'ahbgdc': false
```

`"abc"` matches `a`(0), `b`(2), `c`(5) in order inside `"ahbgdc"`. `"axc"` fails because no `x` ever appears, so the pointer stalls before reaching the end.

## 2. Longest Common Subsequence

### Theory

**Classic example — Longest Common Subsequence (LCS):** find the length of the longest subsequence common to both strings. Build a 2D DP table `dp[i][j]` = LCS length using the first `i` characters of `text1` and first `j` of `text2`: if the current characters match, extend the diagonal (`dp[i-1][j-1] + 1`); otherwise take the best of skipping a character from either string (`max(dp[i-1][j], dp[i][j-1])`). This is the foundational string DP that many later problems (edit distance, shortest common supersequence) build directly on.

### Code

```java
public class LongestCommonSubsequence {
    public static void main(String[] args) {
        System.out.println("LCS length 'abcde'/'ace': " + longestCommonSubsequence("abcde", "ace"));
        System.out.println("LCS length 'abc'/'def': " + longestCommonSubsequence("abc", "def"));
    }

    static int longestCommonSubsequence(String text1, String text2) {
        int m = text1.length(), n = text2.length();
        int[][] dp = new int[m + 1][n + 1];

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }
        return dp[m][n];
    }
}
```

### Example

```
LCS length 'abcde'/'ace': 3
LCS length 'abc'/'def': 0
```

`"ace"` is a subsequence of `"abcde"` (skip `b` and `d`) — length `3`. `"abc"` and `"def"` share no characters at all — length `0`.

## 3. Longest Palindromic Subsequence

### Theory

**Classic example — Longest Palindromic Subsequence:** find the length of the longest subsequence (not substring — characters need not be contiguous) that reads the same forward and backward. Two ways to see it, both useful:

- **Directly:** `dp[i][j]` = longest palindromic subsequence within `s[i..j]`; if the endpoints match, they can both be included around whatever palindrome sits inside (`dp[i+1][j-1] + 2`); otherwise take the best of excluding either endpoint (`max(dp[i+1][j], dp[i][j-1])`).
- **Via LCS:** the longest palindromic subsequence of `s` equals the [Longest Common Subsequence](#2-longest-common-subsequence) of `s` and its own reverse — a neat reduction, since a palindrome read forward is identical to itself read backward.

This directly contrasts with [Lesson 10's Manacher's Algorithm](<./10. Palindromes.md#6-manachers-algorithm>): that finds the longest palindromic **substring** (contiguous) in O(n); this finds the longest palindromic **subsequence** (not necessarily contiguous), which needs O(n²) DP because non-contiguous choices can't be tracked with a simple expanding-center scan.

### Code

```java
public class LongestPalindromicSubsequence {
    public static void main(String[] args) {
        System.out.println("Longest palindromic subsequence, 'bbbab': " + longestPalinSubseq("bbbab"));
        System.out.println("Longest palindromic subsequence, 'cbbd': " + longestPalinSubseq("cbbd"));
    }

    static int longestPalinSubseq(String s) {
        int n = s.length();
        int[][] dp = new int[n][n];

        for (int i = n - 1; i >= 0; i--) {
            dp[i][i] = 1;
            for (int j = i + 1; j < n; j++) {
                if (s.charAt(i) == s.charAt(j)) {
                    dp[i][j] = dp[i + 1][j - 1] + 2;
                } else {
                    dp[i][j] = Math.max(dp[i + 1][j], dp[i][j - 1]);
                }
            }
        }
        return dp[0][n - 1];
    }
}
```

### Example

```
Longest palindromic subsequence, 'bbbab': 4
Longest palindromic subsequence, 'cbbd': 2
```

`"bbbab"` has `"bbbb"` as a length-4 palindromic subsequence (drop the `a`). `"cbbd"` has `"bb"` — length `2` — since no single subsequence longer than that reads the same both ways.

## 4. Counting Distinct Subsequences

### Theory

**Classic example — Distinct Subsequences:** count **how many distinct ways** `t` can be formed as a subsequence of `s` (not just whether it's possible). `dp[i][j]` = number of ways `t[0..j-1]` appears as a subsequence of `s[0..i-1]`. When `s[i-1] == t[j-1]`, that character can either be **used** to match `t[j-1]` (`dp[i-1][j-1]` ways) or **skipped** (`dp[i-1][j]` ways) — both get added; when the characters differ, the character must be skipped (`dp[i-1][j]` only).

### Code

```java
public class DistinctSubsequences {
    public static void main(String[] args) {
        System.out.println("Ways 'rabbit' forms from 'rabbbit': " + numDistinct("rabbbit", "rabbit"));
        System.out.println("Ways 'bag' forms from 'babgbag': " + numDistinct("babgbag", "bag"));
    }

    static int numDistinct(String s, String t) {
        int m = s.length(), n = t.length();
        int[][] dp = new int[m + 1][n + 1];

        for (int i = 0; i <= m; i++) dp[i][0] = 1; // empty t: exactly one way (delete everything)

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                dp[i][j] = dp[i - 1][j]; // always allowed: skip s[i-1]
                if (s.charAt(i - 1) == t.charAt(j - 1)) {
                    dp[i][j] += dp[i - 1][j - 1]; // also allowed: match s[i-1] to t[j-1]
                }
            }
        }
        return dp[m][n];
    }
}
```

### Example

```
Ways 'rabbit' forms from 'rabbbit': 3
Ways 'bag' forms from 'babgbag': 5
```

`"rabbbit"` has three `b`s where `"rabbit"` needs only two consecutive ones — choosing any 2 of the 3 `b`s gives `3` distinct ways.

## 5. Shortest Common Supersequence

### Theory

**Classic example — Shortest Common Supersequence:** find the shortest string that has **both** `s1` and `s2` as subsequences. Any character shared between the two strings (as computed by [LCS](#2-longest-common-subsequence)) only needs to appear **once** in the supersequence, since it can serve as part of both strings' subsequence simultaneously. So the shortest length is simply `len(s1) + len(s2) - LCS(s1, s2)` — the two strings concatenated, minus the double-counted shared subsequence.

### Code

```java
public class ShortestCommonSupersequenceLength {
    public static void main(String[] args) {
        System.out.println("Shortest common supersequence length, 'abac'/'cab': "
                + shortestCommonSupersequenceLength("abac", "cab"));
    }

    static int shortestCommonSupersequenceLength(String s1, String s2) {
        return s1.length() + s2.length() - longestCommonSubsequence(s1, s2);
    }

    static int longestCommonSubsequence(String text1, String text2) {
        int m = text1.length(), n = text2.length();
        int[][] dp = new int[m + 1][n + 1];
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }
        return dp[m][n];
    }
}
```

### Example

```
Shortest common supersequence length, 'abac'/'cab': 5
```

`"abac"` (length 4) and `"cab"` (length 3) share `"ab"` as their LCS (length 2) — shortest supersequence length: `4 + 3 - 2 = 5` (e.g. `"cabac"` contains both `"abac"` and `"cab"` as subsequences).

> Actually **constructing** the supersequence string (not just its length) needs backtracking through the LCS DP table while inserting the non-matching characters from each side — a natural follow-up exercise once the length formula is understood.
