# DSA — Data Structures & Algorithms Notes

A structured collection of notes covering the mathematical foundations, data structures, and algorithms needed for problem solving and interviews.

Every topic follows the same format: **Theory → Java Code → Practical Example**, so you can read the concept, see the implementation, and immediately trace through how it runs.

> **Language used:** Java

---

## Contents

| # | Topic | Description |
|---|-------|-------------|
| 01 | [Maths](./Maths/README.md) | Number theory & arithmetic foundations used across DSA problems |
| 02 | [Arrays](./Arrays/README.md) | Array fundamentals through the full pattern library (prefix/suffix, sliding window, two pointers, monotonic structures, cyclic sort, DP optimizations, and more) |

More topics (Strings, Recursion, Trees, Graphs, DP, etc.) will be added incrementally.

---

## How to use these notes

1. Pick a topic folder from the table above.
2. Open its `README.md` for the topic index.
3. Open the numbered lesson file inside for **Theory + Code + Example**.
4. Copy the Java snippets into your own editor / [Ideone](https://ideone.com) / [JDoodle](https://www.jdoodle.com/online-java-compiler/) and run them yourself — the best way to learn is to tweak the numbers and re-run.
